# Personal Website

This is my personal website, built with [Lume](https://lume.land) and based on the [lumeland/theme-simple-blog](https://github.com/lumeland/theme-simple-blog) repository.

## Features

- Clean and simple blog design
- Supports tags and post organization
- Atom and JSON feeds
- Built-in search functionality
- SEO optimized with sitemap

## About

This website serves as my personal space to share thoughts, projects, and ideas. It's powered by Lume, a static site generator for Deno.

## Development

Built on top of the excellent Simple Blog theme from Lume, customized for my personal needs.
